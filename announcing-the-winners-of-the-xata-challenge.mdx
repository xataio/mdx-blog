## ishn.xyz Link Shortener

Builder: [Ishaan Bedi](https://twitter.com/ishnbedi)

Repository: [github.com/ishaanbedi/ishn.xyz](https://github.com/ishaanbedi/ishn.xyz)

URL: [ishn.xyz](https://www.ishn.xyz/)

ishn.xyz is a fast, feature-rich and completely free link shortener. It has features like custom slugs, statistical tracking, QR Code generation and instant sharing. The Xata team really liked how simple and easy to use this solution was, and that an API was also part of the offering itself. We're really excited to see where Ishaan takes this product suite next.
